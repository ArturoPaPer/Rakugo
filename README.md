Ren is framemork (inspired by [Ren'Py](https://www.renpy.org)) for story driven games in [Godot](https://godotengine.org).

## Story driven games means that are for example:
- Visual Novels (almost done)
- Click & Point Adventures
- RPG (quest system is done)

## You can use it with:
- GDScript (almost done)
- VisualScript (ToDo - I need help)
- RenScript (ToDo)

## Resources / Notes / Links: 
Ren use [PersistenceNode](https://github.com/MatiasVME/Persistence) for saving/load game.
